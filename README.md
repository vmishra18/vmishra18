# About Me

Software engineer focused on building **reliable software systems, APIs, and developer tools**.

My work spans **backend services, mobile applications, analytics dashboards, and automation tooling**, with an interest in designing systems that remain **maintainable, observable, and scalable in production environments**.

I hold a **Master’s degree in Advanced Computer Science** and enjoy exploring how large software systems are designed, deployed, and evolved over time.
